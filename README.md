# 🔐 gaykey

A modern, secure, and minimalist password generator built with **HTML**, **CSS**, and **Vanilla JavaScript**.

Generate strong passwords directly in your browser using the **Web Crypto API**. Nothing is stored, transmitted, or sent to any server.

## ✨ Features

- Secure password generation using `crypto.getRandomValues()`
- Adjustable password length
- Uppercase letters
- Lowercase letters
- Numbers
- Symbols
- Exclude similar characters
- One-click copy to clipboard
- Password strength indicator
- Fully responsive design
- Premium black & beige interface
- Works entirely offline
- No external libraries
